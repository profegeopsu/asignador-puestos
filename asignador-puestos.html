🔗 Guía de configuración — Google Sheets + OAuth

Sigue estos pasos una sola vez. Tiempo estimado: 15 minutos.

PASO 1 — CREAR EL SPREADSHEET
  1. Ve a sheets.google.com y crea un nuevo spreadsheet
  2. Nómbralo AsignadorPuestos
  3. Copia el ID del Spreadsheet desde la URL:
    https://docs.google.com/spreadsheets/d/[ESTE ES EL ID]/edit
  4. Pega ese ID en el campo correspondiente del panel de configuración
PASO 2 — CREAR PROYECTO EN GOOGLE CLOUD
  1. Ve a console.cloud.google.com
  2. Crea un proyecto nuevo → nómbralo AsignadorPuestos
  3. Menú → APIs y servicios → Biblioteca
  4. Busca y activa: Google Sheets API
  5. Menú → APIs y servicios → Credenciales
  6. Clic en Crear credenciales → ID de cliente OAuth
  7. Tipo de aplicación: Aplicación web
  8. Nombre: AsignadorPuestos
  9. En "Orígenes autorizados de JavaScript" agrega la URL donde está alojada la app
    (ej: https://tuusuario.github.io)
  10. Clic en Crear → copia el Client ID que aparece
  11. Pega ese Client ID en el campo correspondiente del panel de configuración
PASO 3 — CONECTAR LA APP
  1. En el panel de Google Sheets de la app, pega el Client ID y el ID del Spreadsheet
  2. Haz clic en Guardar configuración
  3. Haz clic en Conectar con Google → se abrirá el popup de autorización de Google
  4. Autoriza los permisos solicitados (leer y escribir en Sheets)
  5. ¡Listo! Desde ahora los cambios se sincronizan automáticamente
💡 Para GitHub Pages: sube la app a un repositorio público en GitHub, activa Pages en Settings → Pages, y usa esa URL como origen autorizado en Google Cloud.
